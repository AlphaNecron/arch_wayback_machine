# arch_wayback_machine
Download historical Arch Linux packages with one command.

## Dependencies
```
pip install beautifultable htmllistparse
```

## Usage
```
wayback_machine.py package[@version]
                     ^^^     ^^^
                  required  optional
eg: wayback_machine.py neofetch@7.1.0-2
```
